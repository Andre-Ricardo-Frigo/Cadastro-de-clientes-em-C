# Cadastro de Clientes em C

Sistema simples de cadastro de clientes com persistência em arquivo desenvolvido em linguagem C.

---

## Descrição

A aplicação permite cadastrar clientes e armazenar os dados em um arquivo `.txt`. Também possibilita a leitura e listagem dos clientes cadastrados.

---

## Funcionalidades

- Cadastro de clientes (ID, nome e idade)  
- Armazenamento em arquivo  
- Leitura e listagem de dados  
- Suporte a nomes com espaços  
- Estruturação de dados com `struct`  

---

## Tecnologias

- Linguagem C  
- Manipulação de arquivos (`fopen`, `fprintf`, `fscanf`)  
- Estruturas (`struct`)  
- Entrada e saída padrão  

---

## Como executar


### Linux / macOS
```bash
gcc calculadora.c -o calculadora -lm && ./calculadora
```

### Windows (Prompt de Comando)
```bash
gcc calculadora.c -o calculadora.exe -lm && calculadora.exe
```
